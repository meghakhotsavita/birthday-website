
SUPER BIRTHDAY WEBSITE

Features:
- Floating hearts animation
- Cake cutting animation
- Star background
- Photo slider
- Video page
- Wish page
- Background music
- Mobile friendly

How to run:
1. Extract ZIP
2. Add photos in images folder:
   photo1.jpg
   photo2.jpg
   photo3.jpg
3. Add video in video folder:
   birthday_video.mp4
4. Add song in music folder:
   birthday_song.mp3
5. Open index.html

To publish online:
Upload folder to GitHub and deploy using Netlify.
